使用しているソフトを判別†

使用しているソフト（ベースウェア）を判別します。

BwName
{
	if basewarename == "embryo"
	{
		//---- 伺か専用
		"MATERIA"
	}
	elseif basewarename == "SSP"
	{
		//---- SSP専用
		"SSP"
	}
	elseif basewarename == "crow"
	{
		//---- CROW専用
		"CROW"
	}
	else
	{
		//---- その他の場合
		"etc"
	}
}

これを応用して、トークネタを変えたり、機能を制限したりと…。
一工夫してみると、面白いかもしれません。(ひと:猫夢紗)

↑

basewarenameについて†

変数basewarenameは必ずしもベースウェアの名前が入っているという訳ではないため basewarenameをbasewarenameexに書き換えたほうが安全です。
（変数「basewarenameex」ははろーYAYAわーるど、SimpleYAYAテンプレートなどにあります。 他のテンプレートでは存在しない可能性があります）
