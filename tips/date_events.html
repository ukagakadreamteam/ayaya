日付イベント†

テンプレートゴースト「紺野あやめ」のaya_bootend.dic内にある、GetTimeSlot関数を利用すると楽です。

(例)クリスマスイベントの追加
まず、GetTimeSlot内に以下の文を追加します。

elseif month == 12 && day == 25
{
	"クリスマス"
}

次に、OnBoot内にクリスマスの日のみ発動するイベントを追加して完了です。

elseif _timeslot == "クリスマス"
{
	"\1\s[10]\0\s[0]今日はクリスマスです。\e"
}
