shiori.OnLoopLimit†

User-defined function called when the loop limit is reached.

  • The following notes are common to all error handling functions:
    • If you do not need error handling, you do not need to write this function in the dictionary.
    • The return value is ignored.
    • Reference: The GETCALLSTACK function can be used to identify the place where the problem occurred.
↑

Format†

shiori.OnLoopLimit(filename,linenum)

↑

Arguments†

filename
Name of the dictionary file in which the error occurred.
linenum
Line number on which the error occurred.
