shiori.OnMemoryError†

Function to be called on memory error, if defined.

As with other error handling functions:

  • There is no problem if this function is not provided, an error log will simply be generated.
  • By retrieving the call stack with the GETCALLSTACK function, you can determine where the problem occurred.
  • The return value is ignored.
