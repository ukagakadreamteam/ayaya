HEXSTRTOI†

↑

Format†

HEXSTRTOI( string )

↑

Function†

  • 16進数値文字列を整数へ変換します。
    • 頭に0xがついていても無視します。
↑

Arguments†

string
変換したい文字列。
↑

Return value†

  • 成功した場合、変換後の数値
  • 失敗した場合、0
↑

Related†

  • Manual/Function/BINSTRTOI
  • Manual/Function/TOHEXSTR
↑

Version†

  • YAYA：初期から利用可能
    • Tc547-1から、頭に0xがついても大丈夫になった
  • Available in AYA5.8
↑

Example†

_hex = '1E' // 2進数値文字列を代入
_i = HEXSTRTOI( _hex )
_i // 整数「30」 が出力される（16進数の1Eは10進数で30）
