TOUPPER†

↑

Format†

TOUPPER( string [ , locale ] )

↑

Function†

  • 文字列中に含まれる小文字をすべて大文字に変換した文字列を返します。
↑

Arguments†

string
変換したい文字列。
locale
変換時に使用する、setlocaleで使用できるロケール文字列。省略時はC(a-zをA-Zにするだけ)。
↑

Return value†

  • 成功した場合、変換後の文字列
  • 失敗した場合、何も返しません（VOID）
↑

Related†

  • Manual/Function/TOLOWER
↑

Version†

  • YAYA：初期から利用可能 (locale：Tc569-5)
  • Available in AYA5.8
↑

Example†

_str = 'sAkuRa' // 文字列を代入
_str = TOUPPER( _str )
_str // 文字列「SAKURA」 が出力される
