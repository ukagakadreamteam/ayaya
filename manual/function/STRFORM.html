STRFORM†

↑

Format†

STRFORM( format [ , arg ... ] )

↑

Function†

  • 書式化された文字列を返します。
  • C言語のsprintf相当の機能です。
↑

Arguments†

format
書式文字列。
arg
書式文字列の引数。
↑

Return value†

  • 成功した場合、書式化された文字列
  • 失敗した場合の動作は未定義です。
↑

Related†

↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

以下はsprintfとの比較です。結果を戻値で得る点と、書式指定のプレフィックス文字が$である点を除けば、まったく同様に使用できます。

// C言語
int year = 1941;
char *warname = "太平洋";
sprintf(str, "%04d年　%s戦争勃発。", year, warname);
// 文
year = 1941
warname = "太平洋"
str = STRFORM("$04d年　$s戦争勃発。", year, warname)

書式指定一つあたりで展開できる文字列の長さは1024文字までに制限されています。
