RE_GETSTR†

↑

Format†

RE_GETSTR

RE_GETSTR ()

↑

Function†

  • 正規表現系関数の実行結果を取得します。
↑

Arguments†

  • 引数はありません。
↑

Return value†

  • 正規表現内の( )にマッチした部分文字列を列挙した汎用配列を返します。
    • 要素0が正規表現全体にマッチした文字列、要素ｎはｎ番目の( )にマッチした部分文字列です。
  • 正規表現関数を実行していない場合やマッチした文字列が無い場合は IARRAY を返します。
  • RE_REPLACE、RE_REPLACEEX、RE_SPLIT では、( ) ではなく正規表現全体にマッチした部分文字列となります。
↑

Related†

  • Manual/Function/RE_GETPOS
  • Manual/Function/RE_GETLEN
↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

RE_MATCH("x > 15.7", "x *> *([\d\.]+)")
_array = RE_GETSTR

_arrayは("x > 15.7", "15.7")となります。
