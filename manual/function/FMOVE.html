FMOVE†

↑

Format†

FMOVE( path , dir )

↑

Function†

  • ファイルを移動します。
  • パスはフルパス指定可能です。相対パスの場合はDLL loadで渡された位置（多くの場合、DLLのある位置）が基準となります。
↑

Arguments†

path
移動元ファイル名
dir
移動先のディレクトリ名
↑

Return value†

  • 成功した場合、1
  • 失敗した場合、0
