SPLIT†

↑

Format†

SPLIT( src , delim [ , max ] )

↑

Function†

  • 文字列を分割して汎用配列として返します。
  • RE_SPLITで代用可能ですが、正規表現を使わない分だけ高速です。
↑

Arguments†

src
分割したい文字列。（ここに数値型になる可能性のある変数を入れる場合、TOSTRで文字列型変数にするのを推奨）
delim
分割する区切り文字列。
max
最大分割数（省略可能）。0を指定すると全て分割します。省略時は0です。
↑

Return value†

  • 成功した場合、分割後の汎用配列
  • 失敗した場合、空の汎用配列
↑

Related†

  • Manual/Function/RE_SPLIT
↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

_array = SPLIT("A//B//C//D//E", "//", 3)

_arrayは ("A", "B", "C//D//E") となります。
