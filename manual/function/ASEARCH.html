ASEARCH†

↑

Format†

ASEARCH( key , array )

↑

Function†

  • 汎用配列から値をひとつ検索します。
↑

Arguments†

key
検索したいキー値（文字列）。
array
汎用配列（簡易配列は不可）。
↑

Return value†

  • 最初に見つかった要素の序数（先頭＝0）が返ります。
  • 見つからなかった場合は-1が返ります。
↑

Related†

  • Manual/Function/ASEARCHEX
  • Manual/Function/RE_ASEARCH
↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

_array = ('さくら','まゆら','ねここ') // 汎用配列を代入
ASEARCH('まゆら', _array) // 「1」が出力される
