CHRCODE†

↑

Format†

CHRCODE( string [ , pos ] )

↑

Function†

  • 文字列中の１文字のUCS-2コードを返します。
  • UCS-2コードは、ASCII文字（1バイト文字、英数字）の範囲ではASCIIコードと互換性があります。
↑

Arguments†

string
UCS-2コードを知りたい文字列。
pos
コードを知りたい文字の位置（先頭=0）。省略可能。省略時は0が指定されたものとして扱います。
↑

Return value†

  • 成功した場合、変換後の数値
  • 失敗した場合、0
↑

Related†

  • Manual/Function/CHR
↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

CHRCODE('A') // 整数「65」を返す
CHRCODE('ABC', 2) // 整数「67」を返す
CHRCODE('伺') // 整数「20282」を返す
