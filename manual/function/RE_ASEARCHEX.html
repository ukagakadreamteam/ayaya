RE_ASEARCHEX†

↑

Format†

RE_ASEARCHEX( key , array )

↑

Function†

  • 汎用配列から、指定した正規表現に「一部でも」合致する値をすべて検索します。
↑

Arguments†

key
検索したいキー値（文字列）。
array
汎用配列（簡易配列は不可）。
↑

Return value†

  • 見つかった要素の序数を列挙した汎用配列（先頭＝0）が返ります。
  • 見つからなかった場合は空の汎用配列が返ります。
  • RE_GETSTR・POS・LENは使えません。
↑

Related†

  • Manual/Function/ASEARCHEX
  • Manual/Function/RE_ASEARCH
↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

_tea = ("鉄観音", "烏龍", "やぶきた", "烏龍", "ジャスミン")
ipod = RE_ASEARCHEX("^烏", _tea)

ipodには汎用配列 (1,3) が格納されます。
