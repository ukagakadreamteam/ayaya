UNDEFFUNC†

↑

Format†

UNDEFFUNC ( funcname )

↑

Function†

  • 指定した関数を読み込み解除します。
  • 読み込み解除した結果、何か辞書ファイルに問題が起きる場合は、エラーになり何も起こりません。
  • すでに他から参照されている関数の読み込み解除はできません。
↑

Arguments†

funcname
関数名
↑

Return value†

  • 成功した場合、0
  • 失敗した場合、1
