ASEARCHEX†

↑

Format†

ASEARCHEX( key , array )

↑

Function†

  • 汎用配列から値をすべて検索します。
↑

Arguments†

key
検索したいキー値（文字列）。
array
汎用配列（簡易配列は不可）。
↑

Return value†

  • 見つかった要素の序数を列挙した汎用配列（先頭＝0）が返ります。
  • 見つからなかった場合は空の汎用配列が返ります。
↑

Related†

  • Manual/Function/ASEARCH
  • Manual/Function/RE_ASEARCHEX
↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

_tea = ("鉄観音", "烏龍", "やぶきた", "烏龍", "ジャスミン")
ipod = ASEARCHEX("烏龍", _tea)

ipodには汎用配列 (1,3) が格納されます。
