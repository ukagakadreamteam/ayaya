BITWISE_XOR†

↑

Format†

BITWISE_XOR( var1 , var2 )

↑

Function†

  • var1 と var2 のビットXOR演算を行います。
    • 整数（int = 符号付4バイト）に変換し、ビット演算を行います。
↑

Arguments†

var1
数値
var2
数値
↑

Return value†

  • 成功した場合、ビット演算後の数値
  • 失敗した場合、何も返しません。（VOID）
↑

Related†

  • Manual/Function/BITWISE_AND
  • Manual/Function/BITWISE_OR
  • Manual/Function/BITWISE_NOT
  • Manual/Function/BITWISE_SHIFT
↑

Version†

  • TC513-901
↑

Example†

_val1 = 28 // 2進数で11100
_val2 = 9  // 2進数で01001
BITWISE_XOR( _val1, _val2 ) // 整数21が出力される（10101）
