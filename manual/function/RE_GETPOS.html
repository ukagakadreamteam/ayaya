RE_GETPOS†

↑

Format†

RE_GETPOS

RE_GETPOS()

↑

Function†

  • 正規表現系関数の実行結果を取得します。
↑

Arguments†

  • 引数はありません。
↑

Return value†

  • 正規表現内の( )にマッチした部分の位置を列挙した汎用配列を返します。
    • 要素0が正規表現全体にマッチした位置、要素ｎはｎ番目の( )にマッチした部分の位置です。
    • 位置は先頭を0とした文字数です。
  • 正規表現関数を実行していない場合やマッチした文字列が無い場合は IARRAY を返します。
  • RE_REPLACE、RE_REPLACEEX、RE_SPLIT では、( ) ではなく正規表現全体にマッチした部分となります。
