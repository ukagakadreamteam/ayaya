GETTYPEEX†

↑

Format†

GETTYPEEX(var)

↑

Function†

  • Get the value's type.
↑

Arguments†

var
The name of the variable whose type is to be checked, as a string (note that it is not the variable itself).
↑

Return value†

  • 0: Internal error
  • 1: Integer
  • 2: Real number
  • 3: String
  • 4: General array
  • If unsuccessful, 0 is returned.
