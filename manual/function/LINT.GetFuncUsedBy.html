LINT.GetFuncUsedBy†

↑

Format†

LINT.GetFuncUsedBy( function_name )

↑

Function†

  • 指定した関数で使用している関数名を列挙し、汎用配列で返します。
  • ユーザー定義関数・システム関数の両方を列挙します。
↑

Arguments†

function_name
調べたい関数名（文字列）
↑

Return value†

  • 使用している関数名：文字列の汎用配列
  • 候補がない場合は空の汎用配列
  • エラー時は-1
