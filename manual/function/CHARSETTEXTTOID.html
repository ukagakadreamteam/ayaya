CHARSETTEXTTOID†

↑

Format†

CHARSETTEXTTOID( string )

↑

Function†

  • 文字コードを表す文字列を文字コードIDに変換します。
↑

Arguments†

string
文字コードを表す文字列
↑

Return value†

  • 成功した場合、文字コードID
  • 失敗した場合、0
↑

Related†

  • Manual/Function/CHARSETIDTOTEXT
  • マニュアル/文法/B.文字コード関連
↑

Version†

  • YAYA: Available from the start
↑

Example†

CHARSETTEXTTOID('Shift_JIS') // 整数「0」が出力される
CHARSETTEXTTOID('UTF-8') // 整数「1」が出力される
CHARSETTEXTTOID('UTF8') // これも整数「1」が出力される
