LINT.GetGlobalVarLetted†

↑

Format†

LINT.GetGlobalVarLetted( function_name )

↑

Function†

  • 指定した関数で定義された（代入された）グローバル変数名を列挙し、汎用配列で返します。
  • 代入された時点のみを列挙し、読まれる時は無視します。読まれる方も含めて列挙したい場合は LINT.GetGlobalVarUsedBy を使ってください。
↑

Arguments†

function_name
調べたい関数名（文字列）
↑

Return value†

  • 代入されたグローバル変数名：文字列の汎用配列
  • 候補がない場合は空の汎用配列
  • エラー時は-1
