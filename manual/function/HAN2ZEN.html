HAN2ZEN†

↑

Format†

HAN2ZEN( src [ , type ] )

↑

Function†

  • 文字列中に含まれる半角英文字・数字・カタカナをすべて全角に変換します。
↑

Arguments†

src
変換したい文字列。
type
変換したい文字種をカンマ区切りで指定した文字列（省略可能）。省略時は全てを指定したことになります。
  • number - 数値(0～9)
  • alphabet - アルファベット
  • symbol - 記号等
  • kana - カタカナと句読点
↑

Return value†

  • 成功した場合、変換後の文字列
  • 失敗した場合、何も返しません（VOID）
↑

Related†

  • Manual/Function/ZEN2HAN
  • Manual/Function/TRANSLATE
↑

Version†

  • TC513-901
↑

Example†

_result=HAN2ZEN('2007ABC','number,alphabet')

_resultは "２００７ＡＢＣ" となります。
