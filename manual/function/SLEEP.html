SLEEP†

SLEEP( millisecond )

↑

Function†

  • 指定した時間（単位ミリ秒）分実行を中断し待機します。
  • 長い時間を指定するとアプリケーション全体がフリーズします。短時間にとどめるか、デバッグ目的に限定するようにしてください。
↑

Arguments†

millisecond
待機したい時間(ミリ秒)
↑

Return value†

  • 成功した場合、0
  • 失敗した場合、-1
↑
