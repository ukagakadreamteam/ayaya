CHARSETIDTOTEXT†

↑

Format†

CHARSETIDTOTEXT( id )

↑

Function†

  • 文字コードIDを文字コードを表す文字列に変換します。
↑

Arguments†

id
文字コードID（整数）
↑

Return value†

  • 成功した場合、文字コードを表す文字列
  • 失敗した場合、"default"
↑

Related†

  • Manual/Function/CHARSETTEXTTOID
  • マニュアル/文法/B.文字コード関連
↑

Version†

  • YAYA: Available from the start
↑

Example†

CHARSETIDTOTEXT(0) // 文字列「Shift_JIS」が出力される
CHARSETIDTOTEXT(1) // 文字列「UTF-8」が出力される
