ERASE†

↑

Format†

ERASE( src , pos , len )

↑

Function†

  • 文字列の指定部分を削除します。
↑

Arguments†

src
指定部分を削除したい文字列。
pos
削除を開始する文字位置（先頭=0）負値で文字列最後から何文字かの指定に
len
削除する文字数。全体サイズを超えると返り値は常に空文字列に
↑

Return value†

  • 成功した場合、削除後の文字列
↑

Related†

↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
  • Tc539-3でposに負値指定
↑

Example†

_str = 'さくらまゆらねここ'
ERASE( _str, 2, 6 ) // 「さくこ」が出力される
