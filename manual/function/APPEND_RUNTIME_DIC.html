APPEND_RUNTIME_DIC†

↑

Format†

APPEND_RUNTIME_DIC( dic_content )

↑

Function†

  • "_RUNTIME_DIC_" という辞書名のメモリ上の辞書に dic_content の中身を書いたのと同じ効果をもたらします。
  • 動的に新たなユーザ関数定義等が可能です。
↑

Arguments†

dic_content
辞書の中身。複数行・関数定義等を解釈可能。ファイルから辞書を読んだのと同じ挙動になる。
↑

Return value†

  • 成功した場合、0
  • 失敗した場合、-1
