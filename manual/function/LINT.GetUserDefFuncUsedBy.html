LINT.GetUserDefFuncUsedBy†

↑

Format†

LINT.GetUserDefFuncUsedBy( function_name )

↑

Function†

  • 指定した関数で使用しているユーザー定義関数名を列挙し、汎用配列で返します。
↑

Arguments†

function_name
調べたい関数名（文字列）
↑

Return value†

  • 使用しているユーザー定義関数名：文字列の汎用配列
  • 候補がない場合は空の汎用配列
  • エラー時は-1
