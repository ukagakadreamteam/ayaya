TOBINSTR†

↑

Format†

TOBINSTR( var )

↑

Function†

  • 整数を2進数値文字列へ変換します。
↑

Arguments†

var
変換したい整数。
↑

Return value†

  • 成功した場合、変換後の文字列
  • エラーの場合、何も返しません（VOID）
↑

Related†

  • Manual/Function/TOHEXSTR
  • Manual/Function/BINSTRTOI
↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

_i = 30 // 整数を代入
_bin = TOBINSTR( _i )
_bin // 文字列「11110」 が出力される（10進数の30は2進数で11110）
