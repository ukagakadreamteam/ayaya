RE_SPLIT†

↑

Format†

RE_SPLIT ( string , regexp [ , max ] )

↑

Function†

  • 処理対象文字列を正規表現で分割し、分割した内容を汎用配列で返します。
  • 結果の詳細は RE_GETSTR , RE_GETPOS , RE_GETLEN で取得します。
↑

Arguments†

string
対象の文字列。
regexp
正規表現。
max
最大分割数（省略可能）。0を指定すると全て分割します。省略時は0です。
↑

Return value†

  • 成功した場合、分割後の文字列が入った汎用配列。
  • 失敗した場合、空の汎用配列
