LINT.GetLocalVarUsedBy†

↑

Format†

LINT.GetLocalVarUsedBy( function_name )

↑

Function†

  • 指定した関数で使用しているローカル変数名を列挙し、汎用配列で返します。
↑

Arguments†

function_name
調べたい関数名（文字列）
↑

Return value†

  • 使用しているローカル変数名を、使用箇所順に以下の通りの書式で列挙
    • 入れ子(中括弧)内に入る時は { (開き中括弧1文字)
    • 変数を使用していたらその変数名
    • 入れ子(中括弧)内から出る時は } (閉じ中括弧1文字)
  • 候補がない場合は空の汎用配列（中括弧のみの時も空）
  • エラー時は-1
