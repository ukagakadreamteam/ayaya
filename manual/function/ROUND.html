ROUND†

↑

Format†

ROUND( val )

↑

Function†

  • 実数の小数点以下を四捨五入します。
  • 負の値の時の境界値の処理に注意してください。たとえばROUND(-1.55)は-2.0ですが、ROUND(-1.5)は-1.0です！
↑

Arguments†

val
四捨五入したい実数。
↑

Return value†

  • 成功した場合、四捨五入後の実数
  • 失敗した場合、0.0
↑

Related†

  • Manual/Function/CEIL
  • Manual/Function/FLOOR
↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

_val = 1.55 // 1.5以上の実数を代入
ROUND( _val ) // 実数「2.000000」が出力される

_val = 1.45 // 1.5以下の実数を代入
ROUND( _val ) // 実数「1.000000」が出力される

_val = 1.5 // 1.5ちょうどなら…
ROUND( _val ) // 実数「2.000000」が出力される

_val = -1.5 // -1.5ならどうなる？
ROUND( _val ) // 実数「-1.000000」が出力される ※-2.000000 じゃない！

_val = -1.55 // ではこれは？
ROUND( _val ) // 実数「-2.000000」が出力される
