PROCESSGLOBALDEFINE†

↑

Format†

PROCESSGLOBALDEFINE( string )

↑

Function†

  • #globaldefine で定義したプリプロセス処理（文字列置き換え処理）と同様の処理を行います。
↑

Arguments†

string
プリプロセス処理したい文字列
↑

Return value†

  • 成功した場合、stringをプリプロセス処理した文字列
  • 失敗した場合、-1
