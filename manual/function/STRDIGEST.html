STRDIGEST†

↑

Format†

STRDIGEST( string , type )

↑

Function†

  • 指定した文字列のダイジェスト（ハッシュ）値を得ます。
  • 文字列の文字コードは内部表現(UTF-16)になります。
↑

Arguments†

path
ダイジェストを計算したい文字列。
type
ダイジェスト値の種類。"CRC32"で32bitCRC、"MD5"でMD5、"SHA-1"または"SHA1"でSHA-1を得ます。
↑

Return value†

  • 成功した場合、ダイジェスト値の16進数表現文字列
  • 失敗した場合、-1
