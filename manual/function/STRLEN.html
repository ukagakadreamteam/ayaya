STRLEN†

↑

形式†

STRLEN( string )

↑

機能†

  • 文字列の文字数を返します。
  • バイト数では無いことに注意してください（漢字等も１文字と数えます）。
↑

引数†

string
文字数を調べたい文字列。
↑

返り値†

  • 成功した場合、文字数
  • 失敗した場合、0
↑

関連†

↑

バージョン†

  • YAYA：初期から利用可能
  • AYA5.8で利用可能
↑
