TOINT†

↑

Format†

TOINT(string)

↑

Function†

  • Convert strings and real numbers to integers.
↑

Arguments†

string
The string to convert.
↑

Return value†

  • If successful, the converted value.
  • If unsuccessful, 0.
↑

Related†

  • Manual/Function/TOSTR
  • Manual/Function/TOREAL
↑

Version†

  • YAYA: Available from the start
  • Before YAYA Tc542-4 there was a bug that caused it to be processed the same as FLOOR(), but from Tc542-4 onwards the decimal point is simply cut off (the same as in other languages).
  • Available in AYA5.8
↑

Example†

_i = '100' // Assign a string
_j = _i + 1
_j // The string "1001" is output.

_i = '100' // Assign a string
_j = TOINT( _i ) + 1 // Convert to an integer
_j // The integer 101 is output
_i = 1.1 // Assign a real number
_j = TOINT( _i )
_j // The integer 1 is output

_i = -1.1 // What about negative values?
_j = TOINT( _i )
_j // The integer -1 is output ※From Tc542-4 onwards
