FREAD†

↑

Format†

FREAD( path )

↑

Function†

  • ファイルから１行読み取ります。
  • FREADする前にはFOPENでファイルをオープンしておく必要があります。
  • 文字コードはFCHARSETで指定した文字コードからYAYA内部文字コードに変換されます。
  • 行の末尾の改行文字（CR,LF）は取り除かれます。
↑

Arguments†

path
FOPENで指定したファイルへのyaya.dllからの相対パス
↑

Return value†

  • 成功した場合、読み込んだ文字列
  • ファイル終端に達した場合、-1
  • 失敗した場合、空文字列
