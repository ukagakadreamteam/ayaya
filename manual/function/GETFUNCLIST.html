GETFUNCLIST†

↑

Format†

GETFUNCLIST( [ prefix ] )

↑

Function†

  • 現在読み込んでいる辞書の関数リストを返します。
↑

Arguments†

prefix
何らかの文字列を指定すると、その文字列で始まる関数名を返します（省略可能）。省略時は全ての関数のリストを返します。
↑

Return value†

  • 成功した場合、関数名のリスト（汎用配列）
  • 失敗した場合、IARRAY
↑

Related†

  • Manual/Function/GETVARLIST
  • Manual/Function/GETSYSTEMFUNCLIST
↑

Version†

  • YAYA: Available from the start
↑

Example†

//form https://github.com/Taromati2
//free to copy/edit/use
CALLALLFUNCTIONBEGINAS {
	_L= GETFUNCLIST(_argv[0])
	foreach _L;_V {
		EVAL(_V)
	}
}
CALLALLFUNCTIONINRE {
	_L= GETFUNCLIST
	foreach _L;_V {
		if RE_GREP(_V,_argv[0])
			EVAL(_V)
	}
}
