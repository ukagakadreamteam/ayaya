TOREAL†

↑

Format†

TOREAL(string)

↑

Function†

  • 文字列・整数を実数に変換します。
↑

Arguments†

string
変換したい文字列。
↑

Return value†

  • 成功した場合、変換後の数値
  • 失敗した場合、0.0
↑

Related†

  • Manual/Function/TOINT
  • Manual/Function/TOSTR
↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

_s = '1.00005' //文字列を代入
_d = TOREAL(_s)
_d // 実数「1.000050」 が出力される

_i = 1 // 整数を代入
_d = TOREAL( _i )
_d // 実数「1.000000」 が出力される
