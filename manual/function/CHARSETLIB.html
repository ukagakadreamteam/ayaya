CHARSETLIB†

↑

Format†

CHARSETLIB( code )

CHARSETLIB()

CHARSETLIB

↑

Function†

  • 外部ライブラリとのやり取りで使用する文字コードを指定します。
  • LOADLIBの前に実行してください。
  • デフォルトの文字コードは基礎設定ファイルのcharset.extensionエントリ（省略時はcharsetエントリ）で設定した値です。
  • CHARSETLIBでは、全ての外部ライブラリとのやり取りで使用する文字コードを設定します。外部ライブラリ単位で個別に指定したい場合は、CHARSETLIBEXを利用してください。
↑

Arguments†

code
文字コードを文字コードID、または文字コードを示すテキストで指定します。
↑

Return value†

  • codeを指定しなかった場合
    • DLLを呼び出すための文字コードの現在の設定が「文字列で」返ります。
  • codeを指定した場合
    • 戻り値はありません（VOID）
