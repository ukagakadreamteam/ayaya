ASIN†

↑

Format†

ASIN( val )

↑

Function†

  • アークサインをラジアンで返します。
↑

Arguments†

val
アークサインを得たい実数（-1.0 ≦ val ≦ 1.0）
↑

Return value†

  • 成功した場合、アークサイン（ラジアン）
  • 失敗した場合、0.0
↑

↑

Version†

  • YAYA: Available from the start
  • Available in AYA5.8
↑

Example†

_d = ASIN( 0.5 ) // 引数は-1～1の実数
_d // 実数「0.523599」 が出力される（radian）
_d * 180 / 3.14159265358 // 実数「30.000000」が出力される（degree）
